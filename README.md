# Task Manager

A task management web application built with PHP and MySQL.

## Features
- User registration and login
- Create, edit, delete tasks
- Task status: To-Do, In Progress, Done
- Priority levels: High, Medium, Low
- Deadline and labels
- Share tasks with other users
- Notifications system
- Analytics dashboard

## Technologies Used
- PHP 8.2
- MySQL
- HTML/CSS
- XAMPP

## How to Run
1. Install XAMPP
2. Copy project to htdocs folder
3. Import database.sql in phpMyAdmin
4. Open localhost/full_version
